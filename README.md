# 영수증 보관함 (Receipt DB) · v1.16

GPT로 정리한 영수증 텍스트를 영구 보관하고, 가계부처럼 들여다볼 수 있는
**단일 HTML 앱**. 외부 의존성 없음, 오프라인 동작 가능.

🔗 **사용하기**: <https://sh4sh-ux.github.io/receipt-db/>

## 특징

- **OCR/API 없이** ChatGPT·Claude·Gemini 무료 버전으로 사진 → 텍스트 변환 → 앱에 붙여넣기
- **카테고리 자동 분류** — 가게명에서 9개 기본 카테고리(마트/외식/술집/카페/배달/교통/쇼핑/병원·약국/문화) 추정, 사용자가 수정하면 학습
- **가계부 화면** — 월별 총지출, 전월 대비, 카테고리별 비중, 일별 지출 막대
- **모바일·데스크톱 반응형**, 한국어 UI, 깔끔한 화이트 톤 디자인
- **IndexedDB**에 영수증 + 사진 함께 영구 저장
- **JSON 백업** 다운로드/복원 (사진까지 포함)
- **Dropbox 자동 백업** — OAuth 연결 후 버튼 한 번으로 클라우드 백업·복원

## 사용 방법

1. **추가** 탭에서 **프롬프트 복사** 버튼을 누른 뒤,
2. ChatGPT·Claude·Gemini에 영수증 사진과 함께 붙여넣고
3. 받은 응답을 그대로 앱의 붙여넣기 영역에 붙입니다
4. 미리보기 확인 후 사진 첨부 → **저장**
5. **내역** 탭에서 월별 요약 및 영수증 검색

## 백업

브라우저 데이터를 지우면 함께 사라지니, **설정** 탭에서
정기적으로 JSON 백업 파일을 받아두세요. 사진까지 한 파일에 들어갑니다.

## 기술

- 순수 HTML + CSS + Vanilla JS (단일 파일, ~90KB)
- IndexedDB v2 (`receipts` / `images` / `settings` stores)
- 외부 CDN·폰트·빌드 시스템 없음
