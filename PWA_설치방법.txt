# 영수증DB v1.72 PWA 실행 방법

## 가장 쉬운 앱처럼 사용 방법
1. 이 폴더를 GitHub Pages 또는 Netlify에 올립니다.
2. 발급된 웹주소로 접속합니다.
3. Chrome/Edge에서는 주소창 오른쪽 설치 아이콘 또는 앱 설치 버튼을 누릅니다.
4. 아이폰 Safari에서는 공유 버튼 → 홈 화면에 추가를 누릅니다.

## PC에서 임시 테스트
터미널에서 이 폴더로 이동한 뒤 아래 명령어를 실행합니다.

```bash
python3 -m http.server 8000
```

브라우저에서 아래 주소를 엽니다.

```text
http://localhost:8000
```

로컬 파일을 더블클릭하면 PWA 설치가 제한될 수 있습니다.
