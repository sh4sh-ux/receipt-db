영수증DB에 붙여넣을 JSON으로만 답해줘.

규칙:
- 설명, 마크다운, 코드블록 없이 JSON 객체만 출력해줘.
- 상호명, 날짜, 총액, 품목명, 수량, 단가, 금액을 원본 영수증 기준으로 읽어줘.
- 숫자는 쉼표 없이 숫자만 넣어줘.
- 할인/쿠폰/포인트 차감 품목은 금액과 단가를 음수로 넣어줘.
- 수량 x 단가 = 금액이 맞는지 확인해줘. 확실하지 않으면 warnings에 이유를 적어줘.
- 읽기 어려운 값은 추측하지 말고 빈 문자열 또는 0으로 둬.

출력 형식:
{
  "store": "상호명",
  "date": "YYYY-MM-DD",
  "total": 0,
  "items": [
    {
      "name": "품목명",
      "qty": 1,
      "unitPrice": 0,
      "amount": 0,
      "aliases": ""
    }
  ],
  "warnings": []
}
