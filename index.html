앱 설정

모바일·데스크탑 동기화

Supabase 주소와 anon key를 넣으면 휴대전화와 데스크탑이 같은 영수증 DB를 사용합니다. 인터넷이 없으면 로컬에 저장했다가 다시 연결되면 맞춥니다.

연결 안 됨
아직 연결되지 않았습니다. 같은 Supabase 설정과 동기화 비밀번호를 모바일과 데스크탑에 똑같이 넣으면 같은 데이터가 보입니다.

AI 자동 분석 서버

분석 서버 주소를 넣으면 영수증 사진을 OpenAI Vision으로 분석해 입력 초안을 만듭니다. 서버가 없으면 GPT JSON을 직접 붙여넣어도 됩니다.

연결 안 됨

연결 상태

영수증 원본 이미지는 Dropbox에 저장하고, 앱에는 검색 데이터와 작은 미리보기 이미지를 남깁니다.

연결 안 됨

앱에는 검색용 데이터와 작은 썸네일만 두고, 원본 이미지는 Dropbox에 저장하는 구조를 권장합니다.

JSON은 전체 백업/이전용, CSV는 엑셀 확인용입니다. Dropbox 연결 후 저장하면 전체 DB 백업도 Dropbox에 보관하고 다시 불러올 수 있습니다.

Access Token

Dropbox 개발자 콘솔에서 발급한 access token을 입력하면 바로 테스트할 수 있습니다. 실제 배포 버전에서는 서버 OAuth 방식으로 바꾸는 것이 안전합니다.

저장 폴더

기본값은 /영수증DB 입니다. 저장 시 연도/월 폴더가 자동으로 만들어지고 파일명은 250213_영수증(8,000)_매장명.jpg 형식으로 저장됩니다.

Dropbox 연결 후 영수증을 저장하면 원본 확인용 이미지는 Dropbox에 업로드됩니다. 앱 내부에는 화면에서 바로 볼 수 있는 작은 미리보기와 Dropbox 경로가 함께 저장되고, 전체 DB 백업은 backups 폴더에 저장됩니다.
