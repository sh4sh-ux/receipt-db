JSON은 전체 백업/이전용, CSV는 엑셀 확인용입니다. Dropbox 연결 후 저장하면 전체 DB 백업도 Dropbox에 보관하고 다시 불러올 수 있습니다.

Access Token

Dropbox 개발자 콘솔에서 발급한 access token을 입력하면 바로 테스트할 수 있습니다. 실제 배포 버전에서는 서버 OAuth 방식으로 바꾸는 것이 안전합니다.

저장 폴더

기본값은 /영수증DB 입니다. 저장 시 연도/월 폴더가 자동으로 만들어지고 파일명은 250213_영수증(8,000)_매장명.jpg 형식으로 저장됩니다.

Dropbox 연결 후 영수증을 저장하면 영수증 이미지는 최소 용량으로 압축되어 Dropbox에 업로드됩니다. 저장공간을 아끼기 위해 앱 내부에는 이미지 대신 Dropbox 경로가 저장되고, 전체 DB 백업은 backups 폴더에 저장됩니다.
