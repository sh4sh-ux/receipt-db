엑셀로 내보내기

모든 영수증을 스프레드시트 파일(.csv)로 내려받습니다. 엑셀·Numbers·구글 시트에서 바로 열 수 있어요.

JSON 백업 내보내기

모든 영수증과 사진을 하나의 JSON 파일로 내려받습니다. 안전한 곳(Dropbox·iCloud·외장 디스크)에 보관하세요.

JSON 복원

이전에 내려받은 백업 파일을 불러옵니다. 같은 ID의 영수증은 덮어쓰여요.

전체 초기화

저장된 모든 영수증과 사진을 영구히 삭제합니다. 이 작업은 되돌릴 수 없어요. 미리 백업을 받아두세요.

저장 위치

현재 브라우저의 IndexedDB(receiptdb)에 저장됩니다. 브라우저 데이터를 지우면 함께 삭제되니, 정기 백업을 권장합니다.
